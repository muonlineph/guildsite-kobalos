About the Website
The KoB/\Los Guild Website was created by the loyal guild member, tarrent and was first launched online on July 18, 2008. It aims to archive important Guild and Alliance related events, people and memories so that the Members may have something wonderful to reminiesce at as years go by.

Created using hard-coded html tags and javascript, it makes use of the free web hosting services provided by freewebs, the web page redirector freedomains and Google Pages, for uploading files.


Updates for the Website
Only the Editor can update and edit the site, which the Editor promises to constantly update if changes in-game occur, and if time permits. Should any of the Alliance or Guild Members would want to add or post something on any of the pages, please contact or mail the Editor, Tarrent in the mobius forums.


11-May-2010
KoB/\Los Guild Website is BACK online! The KoB/\Los site was down several months after the closing of Google Pages, its main free web host provider. Now, the KoB/\Los Guild Website is fully functional once again after it was renovated to FreeWebs.
